===操作方法===<br>
トリガーボタン：メニューの決定、カメラの向きに移動(動物騎乗時)<br>
バックボタン：メニュー画面に戻る、降りる(動物騎乗時)、降ろす(動物携帯時)<br>


====開発環境===<br>
・Unity 2018.2.14f1
